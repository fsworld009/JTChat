JTChat
===========
自製IRC Client, 設計給Jtv/Twitch實況主抓取聊天室畫面使用
目前還沒有欄位防呆設計，請不要回報因設定欄位格式不合(ex. Port填英文)所產生的錯誤 謝謝

系統需求
===========
Java Runtime Environment (7 or above)
